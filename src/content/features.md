## Core Features

### Drawing Tools

- **Shapes**: Rectangles, circles, and arrows more to come...
- **Lines**: Freehand drawing with smooth curves
- **Text**: Click-to-edit text boxes
- **Selection**: Multi-select with box selection

### Connection System

- **Anchors**: Automatic anchor points on shapes
- **Smart Connectors**: Lines that stay connected when shapes move
- **Visual Feedback**: Highlighted anchor points during connection

### Interaction

- **Transform**: Resize and rotate shapes with transformer handles
- **Drag & Drop**: Move single or multiple elements
- **Box Selection**: Drag to select multiple elements
